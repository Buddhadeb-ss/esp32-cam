# ESP32-CAM Timelapse

A simple ESP32-CAM project that captures JPEG frames to an SD card and
converts them into a timelapse video.

## Demo
[▶ Watch the timelapse](demo/timelapse.mp4)

## How it works
ESP32 → JPEG frames → SD card → FFmpeg → MP4
